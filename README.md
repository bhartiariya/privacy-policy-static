## How to Use

### Local Development

1. Simply open `index.html` in any modern web browser
2. No build process or dependencies required
3. Works completely offline

### Deployment

You can deploy this to any static hosting service:

#### GitHub Pages
1. Push this folder to a GitHub repository
2. Enable GitHub Pages in repository settings
3. Your page will be live at `https://yourusername.github.io/repo-name/`

#### Netlify
1. Drag and drop this folder to [Netlify Drop](https://app.netlify.com/drop)
2. Your site will be instantly deployed

#### Vercel
```bash
cd privacy-policy-static
vercel
```

#### Any Web Server
Simply upload the files to your web server's public directory.

## Customization

### Colors
Edit the CSS custom properties in `styles.css` (lines 10-20):
```css
:root {
    --orange-600: #ea580c;  /* Primary orange */
    --red-600: #dc2626;     /* Primary red */
    /* ... other colors */
}
```

### Content
Edit the HTML content directly in `index.html`. All sections are clearly labeled with comments.

### Animations
To disable animations, remove or comment out the animation properties in `styles.css`, or users can enable "Reduce Motion" in their OS settings.
